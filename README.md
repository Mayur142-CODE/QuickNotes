# 📝 QuickNotes – Android Notes App

**QuickNotes** is a simple and lightweight Android app that lets users create, edit, and delete personal notes. Built with **Java**, **XML**, and **SQLite**, it provides an offline solution for quickly jotting down ideas or reminders with a title and description.

---

## 🚀 Features

- ✅ Add notes with title and description
- ✏️ Edit existing notes
- 🗑️ Delete notes
- 📱 Clean and responsive Android UI
- 💾 Offline storage using SQLite database

---

## 🛠️ Tech Stack

- **Language:** Java
- **UI Design:** XML (Android Layouts)
- **Database:** SQLite
- **IDE:** Android Studio

---

## 📲 Installation & Setup

### ✅ Prerequisites:
- Android Studio installed
- Android device or emulator (API 21+ recommended)
